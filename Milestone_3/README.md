# Milestone 3 – Metagoals & Anti-goals

Milestone 3 connects the M2 metrics into decision scoring by introducing
metagoal adjustments, anti-goal penalties, the overgoal bonus, and the
behaviour-tree planner prototype.

**Core modules**

- `Milestone_3/core/metagoals.metta`
- `Milestone_3/core/antigoals.metta`
- `Milestone_3/core/overgoal.metta`
- `Milestone_3/core/scoring-v2.metta`
