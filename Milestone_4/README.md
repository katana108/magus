# Milestone 4 – Ethical Evaluation & Deliverables

Milestone 4 adds the ethical scenario harness, ablation framework, and the
publication deliverables. The core runtime assets live in `Milestone_4/ethical/`,
`Milestone_4/evaluation/`, and `Milestone_4/tests/`.

**How to run the pipeline**

```bash
source .venv/bin/activate
python Milestone_4/tests/test_m4_pipeline.py
```

**Deliverables**

- Research paper draft: `Milestone_4/docs/MAGUS-Research-Paper-Draft.md`
- Reproducibility package: `Milestone_4/reproducibility-archive/`

Detailed completion summaries, scenario catalogues, and historical notes are
preserved in the knowledge base:
- `magi-knowledge-repo/docs/neoterics/MAGUS/milestones/M4/`
